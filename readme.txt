# MHGCL

# Dependencies
MMGCN is built with Python 3.9.23 with the following packages:
* torch == 2.5.1
* torch-geometric == 2.6.1
* pandas == 2.3.1
* numpy == 1.26.4
* scikit-learn == 1.6.2


# Usage
We provide the scripts for running MHGCL

```
python train.py
```

# Seed
We set global random seed to 2025
